# Level Up HP

RPG style HP upgrades.

## Information

Check out this mod on [CurseForge][].

## Building from source

```bash
git clone https://github.com/sargunv/level-up-hp.git
cd level-up-hp
./gradlew setupDecompWorkspace build runClient
# On Windows, use ./gradlew.bat ...
```
